# Hi, I'm Bianca 👋
**Software Engineer focused on Backend & Platform Engineering**

Building the engineering layer that keeps critical systems reliable, observable, and ready to scale.

I work on backend systems where correctness, consistency, and observability really matter — currently focused on **billing infrastructure**, where the cost of a silent bug is never silent for long.

I moved from frontend → full-stack → backend specialization. Not by accident — I identified backend as a gap in my skill set, made it a focus, sought mentorship, and built until it became a core strength. I'm also expanding into **Python**, adding it alongside Ruby as a tool for backend and automation work.

I care about engineering quality: structured logs, meaningful metrics, clean contracts, and systems that are easy to maintain and understand. And I care about the humans on the team too.
